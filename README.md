# VaultSpin Elite

A fake-money CS-style case simulator with animated case openings, rarity tracking, auto-open queues, and inventory selling.

## Local Run

Open `index.html` in a browser.

## Deploy to Vercel

1. Import this repo in Vercel.
2. Framework preset: `Other`.
3. Build command: none.
4. Output directory: `.` (root).

`vercel.json` is included for SPA-style rewrites.
